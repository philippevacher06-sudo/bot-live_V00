# Journal Codex - Autonomie Technique GPT Temporaire

Date : 2026-05-19
Statut : intervention documentaire publiee GitHub

## Objectif

Eviter l'arret du projet pendant la limite d'usage Codex annoncee jusqu'au
2026-05-23.

## Decision utilisateur

- Du 2026-05-19 au 2026-05-23 inclus, GPT peut agir techniquement si ses outils
  le permettent reellement.
- GPT peut creer/modifier du code, appliquer des patchs, lancer des tests
  disponibles, utiliser un terminal disponible et publier dans GitHub.
- Chaque action technique doit etre journalisee dans GitHub avec preuves,
  commandes/resultats, tests et risques.
- GPT ne doit jamais inventer une action, stocker un secret ou masquer une erreur.
- Broker/ordres restent sous demande explicite separee et preuves de contexte.
