# Amendement V2446 - L3+ et H-001

Date : 2026-05-18
Statut : decision utilisateur a tester avant patch

## Decision

La regle stricte suivante est supprimee :

```text
L3+ direction invalide -> coupe immediate automatique
```

Nouvelle doctrine :

```text
A partir de L3, la verification directionnelle reste obligatoire.
Une rupture M15/M5 ne coupe plus automatiquement le panier pour cette seule raison.
Elle active une gestion de risque stricte.
```

Gestion de risque stricte :

- logs explicites ;
- aucun renfort non valide ;
- respect du maximum de legs ;
- PnL broker prioritaire ;
- stop global ;
- time stop / time slip ;
- verification broker avant patch technique.

## Impact H-001

Cette decision permet de tester `PANIER_CROSS_HEDGE_US500_US100_V1` :

- panier principal US500 L1/L2/L3 ;
- L4 US100 en couverture opposee ;
- pas de L5 ;
- pas de renfort US100 ;
- sortie par PnL broker cumule, stop global ou time slip.

## Verrou

Aucun patch bot n'est autorise tant que les tests offline H-001 et la verification broker ne sont pas prets.
