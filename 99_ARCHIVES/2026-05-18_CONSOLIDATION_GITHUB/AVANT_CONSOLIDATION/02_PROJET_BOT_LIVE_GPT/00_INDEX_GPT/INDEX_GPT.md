Statut : index strategique GPT

## Role

Ce fichier sert d'index pour la memoire strategique GPT du projet `bot-live_V00`.

GPT est l'atelier strategique du projet.

Il travaille sur :

- la doctrine ;
- les arbitrages ;
- les hypotheses ;
- les syntheses ;
- les decisions a valider ;
- les regles a proposer ;
- la preparation eventuelle de consignes vers Codex.

```text
Le chat sert a travailler.
Les fichiers servent a memoriser.
Le pilote sert a router.
```

## 01_DOCTRINE_STRATEGIQUE/

Usage :

- doctrine generale du projet ;
- principes de decision ;
- positionnement strategique ;
- limites a ne pas franchir ;
- coherence entre trading, risque, bot et broker.

## 03_DECISIONS_ET_ARBITRAGES/

Usage :

- arbitrages strategiques ;
- decisions proposees ;
- decisions refusees ;
- raisons de validation ou de refus ;
- consequences attendues.

Une decision validee doit aussi etre reportee dans :

```text
00_COMMUN_SOURCE_DE_VERITE/DECISIONS_VALIDEES.md
```

## 04_SYNTHESES_SESSIONS/

Usage :

- resumes de sessions GPT ;
- reprises de chats longs ;
- points d'avancement ;
- syntheses de bascule vers un nouveau chat.

## 05_HYPOTHESES_A_TESTER/

Usage :

- hypotheses strategiques ;
- hypotheses techniques a transmettre plus tard a Codex ;
- idees non validees ;
- pistes a tester sans decision immediate.

Une hypothese n'est pas une decision.

## 06_REGLES_A_VALIDER/

Usage :

- propositions de nouvelles regles ;
- modifications possibles des regles maitresses ;
- points a faire valider avant inscription dans la source de verite.

Une regle validee doit etre reportee dans :

```text
00_COMMUN_SOURCE_DE_VERITE/BOT_PIVOT_V2446_REGLES_MAITRES.md
```

si elle modifie la doctrine BOT-PIVOT V2446.

## Regle de transmission vers Codex

GPT ne transmet rien a Codex sans paquet explicite valide.

Fichier cible :

```text
04_SYNC_DRIVE/PAQUET_SESSION_GPT_VERS_CODEX.md
```

La phase actuelle ne contient aucune transmission Codex active.

## Principe final

Le role de GPT est de rendre la strategie claire, tracable et transmissible.

Aucune decision importante ne doit rester seulement dans le chat.
